# Documentos del curso

Cada semana tiene su propia carpeta:

```text
public/documentos/semana-01/
public/documentos/semana-02/
...
public/documentos/semana-15/
```

Dentro de cada carpeta coloca estos dos archivos:

```text
ppt-profesor.pdf
trabajo.pdf
```

Ejemplo para la semana 1:

```text
public/documentos/semana-01/ppt-profesor.pdf
public/documentos/semana-01/trabajo.pdf
```

Si un archivo no existe, la web mostrará un estado de "Documento pendiente" en su widget.
